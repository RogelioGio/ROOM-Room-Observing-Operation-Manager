# 🏢 ROOM System
**A Systematic Room, Utility, and Equipment Management Platform**

Built with **Ruby on Rails 8**, **Hotwire**, and **Tailwind CSS v4**.

---

## 📖 Project Overview
The **ROOM System** is a focused, lightweight management platform designed to streamline the administrative overhead of property and utility oversight. 

While purposefully built as a small-scale application, it serves as a technical showcase for **systematic data architecture**. Rather than over-engineering, the project emphasizes high-utility workflows and a polished developer-centric UI.

## 🎯 Technical Highlights
*   **Systematic Management:** A robust, state-driven approach to tracking room occupancy, equipment health, and scheduling.
*   **Maintenance-First CRUD:** A complete suite for staff to manage room equipment and utilities, featuring seamless status editing to ensure digital records reflect physical reality.
*   **Modern Stack:** Leverages **Rails 8** and **Tailwind CSS v4** for a high-performance, future-proof codebase.

## 🛠 Tech Stack
- **Backend:** Ruby 3.3.11 / Rails 8.0
- **Frontend:** Hotwire (Turbo & Stimulus), Tailwind CSS v4
- **Environment:** WSL2 (Ubuntu 24.04)
- **Version Manager:** Mise

## ⚙️ Features
- **Occupancy Dashboard:** At-a-glance monitoring of with status.
- **Utility Tracking:** Manage equipment and inventory within the room inside the building.
- **Equipment Inventory:** Full CRUD capabilities for maintenance personnel to track facility assets and health statuses.
